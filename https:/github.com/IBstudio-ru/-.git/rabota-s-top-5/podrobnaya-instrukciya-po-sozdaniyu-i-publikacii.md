Данная инструкция описывает процесс подготовки, согласования и публикации отзывов к статьям-рейтингам на сайте. Задачу необходимо выполнять **раз в неделю**, при этом дни и время публикаций желательно постоянно чередовать, чтобы активность выглядела естественно.

Актуальный список рейтингов, находящихся в работе, доступен в [сводном файле (Google Docs)](https://www.google.com/search?q=%23).

### Этап 1: Подготовка и генерация черновиков

Для оптимизации процесса рекомендуется делать черновики отзывов пакетами по 5–7 штук на будущее и согласовывать их с руководителем разом. Отзывы пишутся на **все** модели, представленные в рейтинге.

**Специфика контента:**

1. **Модель заказчика:** Отзыв должен быть наиболее позитивным.

2. **Модели конкурентов:** Отзывы должны быть нейтральными. Необходимо подсвечивать некоторые слабые стороны товара, но делать это мягко, без резкой критики.

3. **Стиль:** Живой язык, использование смайликов, легкие и естественные минусы.

**Генерация через нейросети:** Для написания текстов проще всего использовать искусственный интеллект. Поскольку платного доступа сейчас нет, используйте бесплатные варианты (например, Grok или бесплатные версии ChatGPT).

-  *Примечание:* Для работы нейросетей потребуется VPN. Вы можете использовать корпоративный VPN (доступы есть в чате MG) или бесплатные расширения для браузера (например, [это расширение](https://www.google.com/search?q=%23)).

**Промпт (запрос) для нейросети:**

:::quote 

«Привет! Я продолжаю писать отзывы для рейтингов товаров на [mygadget.ru](http://mygadget.ru). Сейчас делаем отзывы на рейтинг *(указать категорию)*. Вот список моделей: *(перечислить модели)*. Нужно по 1 отзыву на каждую модель, один из них более позитивный (для *(указать модель заказчика)*), остальные -- нейтральные. Добавляем смайлики, живой стиль, лёгкие минусы. Примеров из прошлого чата у меня нет -- прошу сразу сформировать заново.»

:::

Все готовые черновики необходимо сохранять в [специальный документ](https://www.google.com/search?q=%23), где они распределены по папкам категорий «Топ-5».

### Этап 2: Поиск и скачивание визуала (фото/видео)

Отзывы преимущественно должны публиковаться с фотографиями. Визуал можно брать из реальных отзывов на маркетплейсе Ozon.

Так как прямое скачивание медиафайлов с Ozon сейчас заблокировано, используйте следующий алгоритм:

1. Откройте нужный отзыв с фото/видео на Ozon.

2. Нажмите правой кнопкой мыши по странице и выберите **«Просмотреть код»** (или «Исследовать элемент»).

3. Нажмите сочетание клавиш **Ctrl + F** для вызова строки поиска по коду.

4. Вбейте в поиск одну из ссылок: [`https://vr-1.ozone.ru`](https://vr-1.ozone.ru) или [`https://v.ozone.ru`](https://v.ozone.ru)

5. Найдите в коде подсвеченную ссылку на медиафайл, скопируйте её и вставьте в новую вкладку браузера.

6. На открывшемся видео или фото нажмите правую кнопку мыши и выберите **«Сохранить как...»**.

### Этап 3: Публикация отзывов на сайте

Блок для комментариев/отзывов находится в самом низу каждой страницы с рейтингом.

**Алгоритм публикации:**

1. **Выбор аккаунта:** Откройте [базу аккаунтов (Google Sheets)](https://www.google.com/search?q=%23). В ней собраны профили (в основном ВКонтакте, но есть и через электронную почту).

2. **Авторизация:** Выберите неиспользованный ранее (или подходящий по графику) аккаунт и авторизуйтесь под ним на сайте.

3. **Публикация:** Зайдите на страницу нужного рейтинга, убедитесь, что система видит вас как авторизованного пользователя, и оставьте подготовленный комментарий с прикрепленным фото.

4. **Ускорение модерации:** Оставленные комментарии не появляются сразу. Чтобы ускорить процесс их публикации, напишите руководителю/модератору в чат, чтобы он оперативно принял и одобрил ваш отзыв через консоль (админку) WordPress.
