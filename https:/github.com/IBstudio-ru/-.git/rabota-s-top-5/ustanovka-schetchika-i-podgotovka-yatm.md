###### Установка счетчика и подготовка ЯТМ

---

## 1\. Установка счетчика Яндекс.Метрики

Перед настройкой целей необходимо создать счетчик и интегрировать его в код страницы на WordPress.



1.  

   **Выбор аккаунта**: Убедитесь, что вы работаете под аккаунтом **ADV** (с него запускаются все проекты «Топ-5»).

2. **Создание счетчика**:

   -   

      **Имя**: Дайте название по заголовку статьи (например, *«Топ-5 портативных колонок до 15 000 ₽»*). Если версий статьи несколько, добавляйте «версия 1», «версия 2».

   -   

      **Адрес**: Укажите URL страницы без протокола (https://).

   -   

      **Настройки**: Включите **Вебвизор** и карту скроллинга. **Электронную коммерцию** и автоматические цели следует **отключить**.

3.  

   **Активация тег-менеджера**: В настройках счетчика обязательно включите галочку **«Тег-менеджер»** и подпункт **«Пользовательский HTML»**, иначе корректная настройка будет невозможна.

4. **Установка кода на сайт**:

   -  Скопируйте полученный код счетчика.

   -  Перейдите в консоль WordPress нужной страницы.

   -  Добавьте блок **«Произвольный HTML»** в самое начало страницы и вставьте туда код.

   -  Сохраните страницу и проверьте статус счетчика в Метрике (строчка должна стать зеленой).

---

## 2\. Настройка главных целей в Метрике

Для корректной оптимизации рекламы требуется создать цель на клик по уникальному идентификатору клиента.



1.  

   **Определение IBS ID**: Выясните последний свободный номер идентификатора (например, `ibs_id=20`).

2. **Создание цели на клик**:

   -  Тип условия: **JavaScript-событие**.

   -  Идентификатор цели: Впишите ваш номер (например, `20`).

   -   

      **Важно**: Скопируйте сгенерированный код цели для дальнейшего использования в тег-менеджере.

3.  

   **Базовая цель**: Сразу добавьте цель типа **«Посещение страницы»**, указав URL статьи.

---

## 3\. Настройка Яндекс.Тег-менеджера (ЯТМ)

ЯТМ используется для автоматической фиксации кликов по ссылкам, содержащим ваш IBS ID.



### Шаг А: Создание триггера

1. Перейдите в раздел «Тег-менеджер» в настройках счетчика.

2. Создайте новый триггер (название: *«Клик по IBS ID=20»*).

3.  

   **Тип условия**: «Клик» -> **«Только ссылки»**.

4.  

   **Условие активации**: Выберите переменную `URL`, условие **«содержит»** и значение вашего ID (например, `20`).

### Шаг Б: Создание тега

1. Создайте новый тег (название: *«IBS 20»*).

2.  

   **Тип тега**: Шаблон **«Пользовательский HTML»**.

3.  

   **Код тега**: Используйте специальный шаблон, вставив внутрь код цели из Метрики.

   -  Код должен быть обернут в теги `<script> ... </script>`.

4.  

   **Привязка**: В нижней части настроек тега привяжите созданный ранее триггер.

### Шаг В: Публикация

1. Нажмите кнопку **«Запустить»**, а затем **«Опубликовать»**.

2.  

   **Помните**: Любые изменения в ЯТМ не вступят в силу на сайте, пока вы не опубликуете новую версию контейнера.

---

## 4\. Проверка и дальнейшие действия

1.  

   **Внедрение ссылок**: После настройки аналитики необходимо заменить все ссылки клиента в статье на ссылки, содержащие `ibs_id=20`.

2.  

   **Тестирование**: Перейдите в статью и сами кликните по ссылкам клиента несколько раз.

3.  

   **Контроль**: Через несколько часов проверьте Метрику -- в отчетах должны появиться данные по цели «Клик по IBS ID». Если данные не отображаются, проверьте, был ли опубликован контейнер в ЯТМ.
